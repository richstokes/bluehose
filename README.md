# bluehose

> Terminal UI for the Bluesky firehose

No real purpose, other than it's kinda fun to watch the messages fly past and count the stats. PR's welcome if you'd like to contribute :-)

## Install

```bash
$ npm install
```

## CLI
```
  Build:
    $ npm run build

  Build & Run:
    $ npm run dev

  Run:
    $ node dist/index.js
```
